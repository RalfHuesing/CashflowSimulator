# Analyse-Charts & MC-Visualisierung

## Beschreibung
Grafische Aufbereitung der Simulationsergebnisse (Avalonia UI).

## Details
- **Percentile-Charts:** Darstellung der Vermögensentwicklung (z.B. 5%, 50%, 95% Perzentile).
- **Interaktiver Pfad:** Auswahl und Detailansicht des Median-Pfades oder anderer spezifischer Läufe.
- **Cashflow-Dashboard:** Monatliche Einnahmen/Ausgaben vs. Liquiditätsreserve visualisieren.
- **Steuer-Analyse:** Visualisierung der gezahlten Steuern über die Zeit.
